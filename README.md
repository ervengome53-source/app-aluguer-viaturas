## Sistema-Aluguer-Viaturas
Sistema para aluguer de viaturas - Projeto de Engenharia de Software II

## Descrição
Sistema desenvolvido para gerenciar o aluguer de viaturas, permitindo cadastro de clientes, gestão de viaturas e controlo de alugueres.

## Funcionalidades
- Cadastro de clientes
- Cadastro de viaturas
- Aluguer de viaturas
- Devolução de viaturas
- Listagem de viaturas disponíveis

## Tecnologias usadas
- PHP
- MySQL
- HTML/CSS
- JavaScript

## Como executar
1. Clonar o repositório
2. Importar a base de dados (database/sistema.sql)
3. Colocar o projeto no servidor (XAMPP ou similar)
4. Acessar pelo navegador
